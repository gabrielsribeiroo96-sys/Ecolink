# Ecolink — Tecnologias e Acesso

**Repositório:** https://github.com/gabrielsribeiroo96-sys/Ecolink  
**Aplicação:** https://ecolink-iota.vercel.app/login

---

## Frontend — JavaScript

| Tecnologia | Função |
|---|---|
| React 19 | Framework de UI |
| Vite 5 | Bundler |
| React Router DOM 7 | Roteamento (SPA) |
| Tailwind CSS 3 | Estilização |
| Radix UI | Componentes acessíveis |
| Leaflet + OpenStreetMap | Mapa interativo (sem API key) |
| Axios | Requisições HTTP |
| React Hook Form + Zod | Formulários e validação |
| Recharts | Gráficos |
| Sonner | Notificações toast |

## Backend — Python

| Tecnologia | Função |
|---|---|
| Python 3.14 | Linguagem |
| FastAPI | Framework web (REST + WebSocket) |
| Uvicorn | Servidor ASGI |
| Pydantic 2 | Validação de dados |
| PyJWT + bcrypt | Autenticação JWT e hash de senha |
| supabase-py | Cliente do banco de dados |
| Starlette CORS | Middleware de CORS |

## Banco de Dados

| Tecnologia | Função |
|---|---|
| Supabase | PostgreSQL gerenciado na nuvem |

## Integrações Externas

| Serviço | Função |
|---|---|
| ViaCEP (`viacep.com.br`) | Autocompletar endereço pelo CEP |
| Nominatim / OpenStreetMap | Geocodificação de endereço para coordenadas |
